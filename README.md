# Amazon GuardDuty

Amazon GuardDuty is an intelligent threat detection service that continuously monitors your AWS accounts, workloads, and data for malicious activity. It uses machine learning, anomaly detection, and integrated threat intelligence to identify and prioritize potential threats to your AWS environment.

## APIs

- **Amazon GuardDuty API** - The Amazon GuardDuty API enables programmatic access to manage threat detection across your AWS accounts. You can create and manage detectors, configure threat intelligence feeds, manage findings, set up member accounts, and create IP sets and threat intelligence lists.
